Live Streaming Camera

一個使用 [StreamPack boilerplate](https://github.com/ThibaultBee/StreamPack-boilerplate) 作底層製作的 Android/Twitch 直播程式。


## 功能

- ✔️ RTMP 推流直播，Twitch 直接推流。
- ✔️ SRT 推流直播。
- ✔️ Twitch 聊天室、直播人數、直播時間。
- ✔️ 藍牙耳機連線。
- ✔️ SRT 直播時顯示資訊，請在設定內開啟。
- ✔️ 全屏顯示，可在設定內開關。
- ✔️ 電池資訊、溫度顯示，可在設定內開關。
- ✔️ Wifi、Sim卡訊號顯示，可在設定內開關。
- ✔️ 左右音量顯示，可在設定內開關。
- ✔️ 手機抖動顯示，可在設定內開關。
